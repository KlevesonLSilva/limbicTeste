Apresentação do Projeto Este é um pequeno projeto de teste desenvolvido para a empresa Limbic, utilizando Angular versão 18 e Angular Material versão 18.2.12, juntamente com TailwindCSS.
Funcionalidades Implementadas Sidebar: Uma barra lateral que exibe o logotipo da empresa e duas opções de navegação: "Proyectos" e "Cerrar Sesión". Proyectos: Ao clicar em "Proyectos", uma barra de progresso é exibida, indicando o status atual do processo. Além disso, um input é disponibilizado para carregar arquivos no formato .xlsx. Carregar Arquivo: O input permite selecionar um arquivo .xlsx do sistema de arquivos. Caso o arquivo selecionado não seja do tipo .xlsx, uma mensagem de erro é exibida. Se o arquivo for válido, ele é carregado e uma tabela com as informações do arquivo é exibida abaixo. Botões de Navegação: Dois botões são fornecidos para facilitar a navegação: "Voltar" e "Finalizar".
Tecnologias Utilizadas Angular v18: Framework utilizado para o desenvolvimento da aplicação. Angular Material v18.2.12: Biblioteca de componentes de interface de usuário utilizada para estilização e componentes prontos. TailwindCSS: Framework de CSS utilizado para estilização personalizada.
